Plugins

Yarn supports plugins. This allows you to plug into lifecycle events during Yarn's build process.

See all events currently supported.

Current Yarn Plugins

  • yarn-excerpt

Add excerpt data to each File object. Useful when rendering a list of pages.

  • yarn-html-minifier

Minify your rendered files.
